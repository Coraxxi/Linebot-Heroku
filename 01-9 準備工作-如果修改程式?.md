```
         D:

                |__ package.json
```

## 1. 通常會改index.js

## 2. 再將本地端應用程式上傳至Heroku應用程式

```
(1) heroku login
(2) git init
(3) heroku git:remote -a [Heroku上的應用程式名稱]
---------------------------------------------------
(4) git add .
(5) git commit -am "myApp"
(6) git push heroku master -f
```
