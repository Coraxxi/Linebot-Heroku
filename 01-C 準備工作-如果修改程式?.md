## 2. 再將本地端應用程式上傳至Heroku應用程式
```
如果(1)~(7)已完成, 不必每次執行, 以後執行(8)~(10)即可.
```

```
(1) (網頁)已登入Github
(2) (網頁)已登入Line Developer
(3) (網頁)已登入Heroku
(4) heroku login
(5) git config --global user.email "自己在git的email帳號"
(6) git init
(7) heroku git:remote -a [Heroku上的應用程式名稱]
---------------------------------------------------
(8) git add .
(9) git commit -am "myApp"
(10) git push heroku master -f
```
